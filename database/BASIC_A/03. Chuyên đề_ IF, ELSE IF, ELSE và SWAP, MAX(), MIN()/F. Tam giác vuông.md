# F. Tam giác vuông

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào ba số nguyên dương $a, b, c$. In chữ "YES" nếu 3 số là chiều dài của 3 cạnh tam giác vuông, ngược lại in chữ "NO".

### Input
- Ba số nguyên dương $1 \le a, b, c \le 10^9$.

### Output
- In ra YES/NO.

### Examples
| input | output |
| :--- | :--- |
| 3 4 5 | YES |
| 4 5 4 | NO |
