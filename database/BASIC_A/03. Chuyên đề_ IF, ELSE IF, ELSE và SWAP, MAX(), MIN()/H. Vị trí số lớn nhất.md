# H. Vị trí số lớn nhất

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên $x$ có 3 chữ số khác nhau. In ra số lớn nhất và nằm ở hàng nào?

### Input
- Một số nguyên dương có 3 chữ số khác nhau.

### Output
- Chữ số lớn nhất và vị trí (TRAM, CHUC, DONVI).

### Examples
| input | output |
| :--- | :--- |
| 123 | 3 DONVI |
| 765 | 7 TRAM |
| 453 | 5 CHUC |
