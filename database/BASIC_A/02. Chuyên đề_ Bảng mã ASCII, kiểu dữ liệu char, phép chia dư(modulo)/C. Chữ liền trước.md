# C. Chữ liền trước

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một chữ cái in hoa $c$, in ra chữ cái thường đứng liền trước chữ cái in hoa $c$.

### Input
- Một chữ cái $c$.

### Output
- Một chữ cái thỏa mãn yêu cầu.

### Examples
| input | output |
| :--- | :--- |
| F | e |
| E | d |
