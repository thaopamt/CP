# B. Ký tự cũ

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào chữ cái thường, in ra chữ in hoa tương ứng.

### Input
- Một ký tự là chữ cái thường.

### Output
- In ra chữ in hoa tương ứng.

### Example
| input | output |
| :--- | :--- |
| a | A |
