# B. Đếm dấu khoảng trắng

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Cho một chuỗi kí tự $s$ có $n$ kí tự ($n \le 100$). Hãy đếm số kí tự khoảng trắng trong chuỗi đó.

### Input
- Gồm một dòng duy nhất là chuỗi kí tự $s$.

### Output
- In ra số lượng kí tự khoảng trắng của $s$.

### Example
| input | output |
| :--- | :--- |
| kid 1 4 1 2 | 4 |
