# A. Chênh lệch độ dài

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Cho 2 chuỗi kí tự $a$ và $b$. Hãy in ra độ chênh lệch độ dài của 2 chuỗi.

### Input
- Dòng thứ nhất là chuỗi kí tự $a$.
- Dòng thứ hai là chuỗi kí tự $b$.

**Lưu ý:** Chuỗi nhập vào có thể có dấu khoảng trống (nên dùng `getline(cin, s)` thay vì `cin >> s`).

### Output
- Gồm một dòng duy nhất là kết quả cần tìm.

### Example
| input | output |
| :--- | :--- |
| zzzzzz aa<br>ssssss aaaaaa | 4 |
