# C. Đếm ký tự trong xâu

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Cho một xâu ký tự $s$ gồm các ký tự trong bảng mã ASCII và một ký tự $c$.

### Yêu cầu
Hãy đếm số lần ký tự $c$ xuất hiện trong xâu $s$.

### Input
- Dòng 1 chứa xâu $s$ ($|s| < 5000$).
- Dòng 2 chứa ký tự $c$.

### Output
- Số lần ký tự $c$ xuất hiện trong xâu $s$.

### Example
| input | output |
| :--- | :--- |
| danang<br>a | 2 |
