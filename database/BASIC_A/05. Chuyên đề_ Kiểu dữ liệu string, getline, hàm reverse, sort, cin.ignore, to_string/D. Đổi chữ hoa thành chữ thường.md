# D. Đổi chữ hoa thành chữ thường

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Cho một chuỗi kí tự gồm $n$ kí tự bất kì ($n \le 100$). Hãy đổi tất cả chữ hoa có trong chuỗi thành chữ thường. Xuất chuỗi ra màn hình.

### Input
- Gồm một dòng duy nhất là một chuỗi kí tự.

### Output
- In chuỗi đã đổi ra màn hình.

### Example
| input | output |
| :--- | :--- |
| 4I1K2D14Ti | 4i1k2d14ti |
