# A. Số chính phương

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên $n$. Kiểm tra xem $n$ có phải là số chính phương hay không? (Số chính phương là bình phương của một số nguyên như $16 = 4^2$).

### Input
- Một số nguyên dương $n$ ($1 \le n \le 10^{18}$).

### Output
- Là số chính phương thì in `YES`, ngược lại in `NO`.

### Examples
| input | output |
| :--- | :--- |
| 4 | YES |
| 100 | YES |
| 123 | NO |
