# B. Hình vuông số tăng dần theo hàng

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($n \le 100$). Hãy in ra bảng số có $n$ dòng, mỗi dòng in $n$ số từ 1 đến $n$.

### Input
- Một số nguyên dương $n$ ($n \le 100$).

### Output
- Bảng số theo yêu cầu.

### Example
| input | output |
| :--- | :--- |
| 4 | 1 2 3 4<br>1 2 3 4<br>1 2 3 4<br>1 2 3 4 |
