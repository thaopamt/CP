# I. Tổng dãy số tự nhiên liên tiếp

- **Time limit per test:** 0.25 seconds
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên $x$. Hãy xác định xem $x$ có phải tổng của các số tự nhiên liên tiếp bắt đầu từ 1 hay không?

### Input
- Một số nguyên dương $x$ ($1 \le x \le 10^{18}$).

### Output
- In ra YES/NO.

### Examples
| input | output |
| :--- | :--- |
| 10 | YES |
| 14 | NO |
