# C. Hình vuông số tăng dần nhưng ảo hơn

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($n \le 100$). Hãy in ra bảng số có $n$ dòng, mỗi dòng in $n$ số nguyên liên tiếp, dòng thứ $i$ bắt đầu từ số $i$.

### Input
- Một số nguyên dương $n$ ($n \le 100$).

### Output
- Bảng số theo yêu cầu.

### Example
| input | output |
| :--- | :--- |
| 4 | 1 2 3 4<br>2 3 4 5<br>3 4 5 6<br>4 5 6 7 |
