# D. Hình tam giác số tăng dần

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($n \le 100$). Hãy in ra bảng số có $n$ dòng, dòng thứ $i$ in ra $i$ số tự nhiên liên tiếp bắt đầu từ số $i$.

### Input
- Một số nguyên dương $n$ ($n \le 100$).

### Output
- Bảng số theo yêu cầu.

### Example
| input | output |
| :--- | :--- |
| 4 | 1<br>2 3<br>3 4 5<br>4 5 6 7 |
