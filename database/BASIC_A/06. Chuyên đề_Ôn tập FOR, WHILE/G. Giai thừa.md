# G. Giai thừa

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Cho một số nguyên $n$ ($1 \le n \le 10$). Hãy tính giá trị của $n$ giai thừa. Giai thừa của 1 số nguyên $n$ là tích của tất cả các số nguyên từ 1 đến $n$.

### Input
- Gồm 1 số nguyên $n$ duy nhất.

### Output
- Giá trị của $n$ giai thừa.

### Example
| input | output |
| :--- | :--- |
| 3 | 6 |

### Note
Giải thích: $3! = 1 \times 2 \times 3 = 6$.
