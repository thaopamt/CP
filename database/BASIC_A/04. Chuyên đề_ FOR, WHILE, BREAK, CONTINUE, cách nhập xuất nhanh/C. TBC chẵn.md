# C. TBC chẵn

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($n \le 10^6$). Hãy in ra trung bình cộng các số chẵn từ 1 đến $n$.

### Input
- Một số nguyên dương $n$.

### Output
- Một số thực (hoặc nguyên nếu đáp án là số nguyên) là trung bình cộng các số chẵn từ 1 đến $n$.

### Example
| input | output |
| :--- | :--- |
| 17 | 9 |
