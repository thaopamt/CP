# I. Hình chữ nhật rỗng

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào hai số nguyên dương $n, m$ ($3 \le n, m \le 25$). Hãy in ra hình chữ nhật rỗng có kích thước $n \times m$.

### Input
- Hai số nguyên dương $n, m$.

### Output
- Hình chữ nhật rỗng dấu *.

### Example
| input | output |
| :--- | :--- |
| 3 5 | \*\*\*\*\*<br>\*&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;\*<br>\*\*\*\*\* |
