# H. Hình vuông rỗng

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($3 \le n \le 25$). Hãy in ra hình vuông rỗng có kích thước $n \times n$.

### Input
- Một số nguyên dương $n$.

### Output
- Hình vuông rỗng dấu *.

### Example
| input | output |
| :--- | :--- |
| 3 | \*\*\*<br>\*&nbsp;&nbsp;\*<br>\*\*\* |
