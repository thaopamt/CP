# J. Hình tam giác đặt

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($3 \le n \le 25$). Hãy in ra hình tam giác vuông đặt có $n$ dòng, với mỗi dòng, dòng thứ $i$ có $2 \times i - 1$ dấu *.

### Input
- Một số nguyên dương $n$.

### Output
- Hình tam giác đặt dấu *.

| input | output |
| :--- | :--- |
| 4 | \*<br>\*\*\*<br>\*\*\*\*\*<br>\*\*\*\*\*\*\* |
