# F. Hình vuông đặt

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên dương $n$ ($n \le 25$). Hãy in ra hình vuông đặt có $n$ dòng, mỗi dòng có $n$ dấu *.

### Input
- Một số nguyên dương $n$.

### Output
- Hình vuông đặt dấu *.

### Example
| input | output |
| :--- | :--- |
| 3 | \*\*\*<br>\*\*\*<br>\*\*\* |
