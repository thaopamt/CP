# E. Số lượng ước số của n

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào số nguyên $n$ ($n \le 10^{12}$). In ra số lượng ước số của $n$. (Ước số của $n$ là các số nguyên mà $n$ chia hết).

### Input
- Một số nguyên dương $n$.

### Output
- In ra số lượng ước số của $n$.

### Examples
| input | output |
| :--- | :--- |
| 10 | 4 |
| 36 | 9 |

### Note
Ví dụ n = 10 thì 4 ước số: 1, 2, 5, 10.
