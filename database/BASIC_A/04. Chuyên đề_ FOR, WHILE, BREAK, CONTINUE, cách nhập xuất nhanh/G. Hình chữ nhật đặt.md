# G. Hình chữ nhật đặt

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào hai số nguyên dương $n, m$ ($n, m \le 25$). Hãy in ra hình chữ nhật đặt có $n$ dòng, mỗi dòng có $m$ dấu *.

### Input
- Hai số nguyên dương $n, m$.

### Output
- Hình chữ nhật đặt dấu *.

### Example
| input | output |
| :--- | :--- |
| 3 5 | \*\*\*\*\*<br>\*\*\*\*\*<br>\*\*\*\*\* |
