# F. Số lượng số hạng #2

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào một số nguyên $N$, in ra màn hình số lượng số dương nhỏ hơn hoặc bằng $N^2$ và lớn hơn hoặc bằng $N+1$.

### Input
- Số nguyên $N \le 200$.

### Output
- In ra số nguyên thỏa mãn.

### Example
| input | output |
| :--- | :--- |
| 3 | 6 |
