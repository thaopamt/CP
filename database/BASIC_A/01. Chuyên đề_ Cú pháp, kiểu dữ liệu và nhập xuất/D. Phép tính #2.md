# D. Phép tính #2

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào hai số nguyên, in ra kết quả của phép cộng (+), phép trừ (-), phép nhân (*), phép chia (/). Nhận xét kết quả chia 2 số nguyên.

### Input
- Hai số nguyên.

### Output
- Dòng 1: kết quả phép cộng.
- Dòng 2: kết quả phép trừ.
- Dòng 3: kết quả phép nhân.
- Dòng 4: kết quả phép chia.

### Example
| input | output |
| :--- | :--- |
| 8 4 | 12<br>4<br>32<br>2 |
