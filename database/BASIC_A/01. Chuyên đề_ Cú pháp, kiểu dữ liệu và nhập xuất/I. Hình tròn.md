# I. Hình tròn

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào số thực $R$ của bán kính hình tròn, in ra chu vi và diện tích hình tròn (lấy 1 số lẻ).

$pi = 3.14$

Công thức chu vi: $2 \times r \times pi$

Công thức diện tích: $r \times r \times pi$

### Input
- Số thực $R$.

### Output
- Dòng 1: Chu vi.
- Dòng 2: Diện tích.

### Example
| input | output |
| :--- | :--- |
| 1 | 6.3<br>3.1 |
