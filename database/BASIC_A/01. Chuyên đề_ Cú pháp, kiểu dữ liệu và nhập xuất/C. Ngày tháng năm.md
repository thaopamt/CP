# C. Ngày tháng năm

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào ba số nguyên biểu diễn ngày tháng năm, in ra ngày tháng năm dưới dạng *ngay / thang / nam*.

### Input
- Ba số nguyên.

### Output
- *ngay / thang / nam*.

### Example
| input | output |
| :--- | :--- |
| 12 7 2017 | 12/7/2017 |
