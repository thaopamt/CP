# B. Phép tính #1

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Viết chương trình nhập vào 3 số nguyên $A, B$ và $C$. Hãy in ra màn hình giá trị biểu thức $(A-B) \times C$.

### Input
- Ba số nguyên $1 \le A, B, C \le 10$.

### Output
- In ra số nguyên là đáp án.

### Example
| input | output |
| :--- | :--- |
| 10 1 3 | 27 |
