# D. Bé hơn

- **Time limit per test:** 1 second
- **Memory limit per test:** 256 megabytes

Cho dãy số nguyên dương gồm $n$ phần tử $a_1, a_2, \dots, a_n$. Với mỗi chỉ số $1 \le i \le n$, đếm xem có bao nhiêu phần tử bé hơn $a_i$.

### Input
- Dòng đầu tiên gồm số nguyên dương $n$ ($2 \le n \le 10^5$).
- Dòng thứ hai gồm $n$ số nguyên dương $a_1, a_2, \dots, a_n$ ($a_i \le 10^9$).

### Output
- In ra $n$ số nguyên, số thứ $i$ cho biết số phần tử nhỏ hơn $a_i$.

### Example
| input | output |
| :--- | :--- |
| 5<br>3 2 1 1 2 | 4 2 0 0 2 |
